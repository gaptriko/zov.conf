## ✨ Основные возможности

*   ⚡️ **Бесшовный доступ:** Автоматически направляет заблокированные ресурсы (Gemini, ChatGPT, YouTube, Instagram, X/Twitter, Facebook, GitHub) через ваши прокси-серверы.
*   🇷🇺 **Прямой маршрут (DIRECT):** Российские сайты и сервисы (Госуслуги, банки, Авито, WB, Ozon, Яндекс, Кинопоиск) работают напрямую, сохраняя максимальную скорость и оригинальный IP.
*   🔒 **Защита от утечек DNS:** Оптимизированные настройки DNS используют системные сервера и защищенные DoH (DNS-over-HTTPS) от Cloudflare и Google.
*   📱 **Энергоэффективность:** Правила скомпилированы так, чтобы минимизировать нагрузку на аккумулятор вашего устройства при постоянной работе VPN.

## 🛠 Таблица маршрутизации трафика

Вся сеть автоматически разделяется на три основных направления:

| Категория ресурсов | Примеры сервисов | Направление трафика |
| :--- | :--- | :---: |
| 🤖 **ИИ & Разработка** | Gemini, OpenAI, ChatGPT, GitHub | 🌐 `PROXY` |
| 🎬 **Соцсети & Видео** | YouTube, Instagram, X (Twitter), TikTok | 🌐 `PROXY` |
| 💬 **Мессенджеры** | Telegram (включая звонки и загрузку медиа) | 🌐 `PROXY` |
| 🏛 **Государственные службы** | Госуслуги, nalog.ru, официальные порталы | 🔌 `DIRECT` |
| 💳 **Банки & Доставки** | Сбербанк, Т-Банк, WB, Ozon, Авито, Самокат | 🔌 `DIRECT` |
| 🏠 **Локальные сети** | 192.168.x.x, 10.x.x.x, localhost | 🔌 `DIRECT` |
| 🌎 **Все прочие домены** | Любые другие не описанные ресурсы | 🌐 `PROXY` |

<details>
<summary>🔍 Дополнительные технические особенности (кликните для раскрытия)</summary>

*   **Bypass System:** Конфиг спроектирован с поддержкой опции `bypass-system = true`, что автоматически пускает системный трафик Apple в обход прокси для экономии трафика и стабильности iOS.
*   **QUIC/HTTP3:** Трафик UDP на 443 порт по умолчанию отклоняется (`REJECT-NO-DROP`), чтобы заставить приложения (например, YouTube) переключиться на TCP, что обеспечивает более стабильное проксирование.
</details>

## 📲 Инструкция по настройке на iOS / macOS

1. Скопируйте ссылку на файл `zov.conf`.
2. Откройте приложение **Shadowrocket**.
3. Перейдите во вкладку **Config** (Конфигурация) в нижнем меню.
4. Нажмите кнопку **`+`** в правом верхнем углу экрана.
5. Вставьте скопированную ссылку в поле ввода и нажмите **Download**.
6. Выберите скачанный файл `zov.conf` в списке и нажмите **Use Config** (Использовать конфигурацию).
7. На главной вкладке приложения установите режим **Global Routing** (Глобальная маршрутизация) в положение **Config**.
